cách tạo web giới thiệu bản thân không cần code

bạn chỉ cần bỏ ra:

+ 500,000Cowoncy để tạo web với mẫu có sẵn.

+ 800,000Cowoncy để tạo web tự code (tức là bạn sẽ gửi code html, css, js cho chúng mình để chúng mình up web lên).

● web ở đây sẽ là web tĩnh và chỉ có một page.

● dưới dây là các bước để tạo một trang web:

+ bạn có thể dms trực tiếp với mình đề bàn bạc nhưng mình khuyên bạn nên vào máy chủ discord ấn để vào và vào kênh ấn dễ xem và tạo vé để bàn bạc.

+ đối với mẫu tự code bạn chỉ cần gửi file: html, css, và js cho mình và gửi tên web bạn muốn đặt, nếu mình online được mình sẽ up web lên.

+ đối với mẫu có sẵn bạn phải làm theo các bước sau:

1. đặt tên web: chọn tên web bạn muốn và gửi cho mình.

2. viết description của link nhúng (có thể bỏ qua, nếu bỏ qua link nhúng description mặc định sẽ là "web của {tên tài khoản discord của bạn}").

3. avatar của bạn:

● có 2 cách lấy avatar:

+ bạn có thể gửi ảnh cho mình (có thể gửi ảnh png hay gif, gif đừng có nặng quá).

+ đưa url ảnh cho mình, đảm bảo url không lỗi nếu không avatar sẽ lỗi (có thể bỏ qua).

4. tên của bạn: bạn có thể gửi tên của bạn cho mình.

5. nội dung chính:

● bạn có thể viết nội dung giới thiệu về mình hay giới thiệu gì đó cũng được.

● bạn có thể chọn cỡ chữ cho nội dung, có 6 loại cỡ chữ 1 là to nhất, 6 là nhỏ nhất.

● cách trình bày:

+ tiêu đề: bạn sẽ sử dụng kí hiệu cỡ chữ" 1 đến 6 (nội dung bạn muốn hiên lên web) "để chúng mình hiểu. ví dụ như:

1 (tiêu đề 1)

2 (tiêu đề 2)

6 (tiêu đề 3) (có thể bỏ qua).

+ nội dung: bạn sẽ sử dụng kí hiệu" (nội dung bạn muốn hiện lên web) "để chúng mình hiểu. ví dụ như:

(nội dung 1)

(nội dung 2)

(nội dung 3)

+ link ẩn: bạn sẽ sử dụng kí hiệu" [ấn để vào máy chủ](url thích hợp) "để chúng mình hiểu. ví dụ như:

hãy [ấn vào đây](https://dsc.gg/hrv-rero-clan) để vào máy chủ của tôi. (có thể bỏ qua).

7. hình ảnh:

● cung cấp cho mình hình ảnh, bạn có thể cung cấp url hoặc gửi ảnh trực tiếp cho mình (có thể bỏ qua).

8. liên kết mạng xã hội:

● bạn có thể gửi cho mình các liên kết mạng xã hội của bạn như: youtube, tiktok, discord (có thể bỏ qua).

9. backgroud:

● có 2 loại backgroud (ảnh nền).

+ backgroud color: bạn chỉ cần cung cấp mã hex cho mình là được, mã hex sẽ trông như: #20ff00 (có thể bỏ qua).

+ backgroud iamge: bạn chỉ cần gửi ảnh, hoặc cung cấp ảnh cho mình là được.

10. màu chữ: bạn có thể cung cấp cho mình màu chữ bạn yêu thích dưới dạng mã hex. ví dụ như #000000 (có thể bỏ qua).

lưu ý nhỏ: chúng mình sẽ không nhận làm web có nội dung lừa đảo, nói xấu người khác, nội dung quá dark và tên web của bạn phải không được chứa các kí tự đặc biệt (trừ dấu "-" có thể dùng dể cách chữ trong tên trang web) và khi viết nôi dung web cần lưu ý chính tả vì để sửa web thì bạn vó thể mất đến 1 - 3 ngày thì web mới được cập nhật.
